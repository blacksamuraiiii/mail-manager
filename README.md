# 邮件管家

## 项目描述

这是一个用于管理邮件的Python项目。

## 安装

1. 克隆仓库：

   ```bash
   git clone https://github.com/blacksamuraiiii/mail-manager.git
   cd mail-manager
   ```
2. 安装依赖：

   ```bash
   pip install -r requirements.txt
   ```

## 使用方法

运行 `main.py` 文件：

```bash
python main.py
```

## 配置文件

请在 `config.py` 中配置您的邮件服务器信息和凭据。
